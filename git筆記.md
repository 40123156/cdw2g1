=========================== 更動檔案前 ===========================

設定 git config

`git config --global user.name "name"`
<br />
`git config --global user.email name@example.com`

將資料 clone 至近端

`git clone [--]`

如果已經近端已有資料, 想要更新可用 pull

`git pull`

切換分支指令

`git checkout [--]`

=========================== 更動檔案後 ===========================

`git add -A`

`git commit -m "message"`

`git push`
