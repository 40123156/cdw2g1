=========================== 更動檔案前 ===========================

設定 git config

`git config --global user.name "name"

git config --global user.email name@example.com`

`git clone [--]`

切換分支指令

`git checkout [--]`

=========================== 更動檔案後 ===========================

`git add -A`

`git commit -m "message"`

`git push`
